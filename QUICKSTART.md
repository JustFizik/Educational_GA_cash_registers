# Быстрый старт - Оптимизация расписания кассиров

## За 5 минут

### Шаг 1: Установка зависимостей

Установите необходимые зависимости для рабюоты прокета, если они отсутствуют в Вашей системе.

### Шаг 2: Запуск Jupyter Notebook

```bash
jupyter notebook CassaScheduler.ipynb
```

или

```bash
jupyter lab CassaScheduler.ipynb
```

### Шаг 3: Запуск алгоритма

В открывшемся Notebook выполните: **Cell → Run All**

### Шаг 4: Получение результатов

После завершения работы алгоритма (обычно 1-3 минуты) Вы получите:

1. **Консольный вывод** с расписанием
3. **JSON файл** `best_schedule.json` с детальными результатами
4. **PNG файл** `schedule_visualization.png` с графиками

## Настройка под Ваши данные

### Изменение времени работы:

```python
WORK_START_HOUR = 6   # Ваше время начала
WORK_END_HOUR = 23    # Ваше время окончания
```

### Изменение потока клиентов:

```python
CLIENT_FLOW = [100, 120, 100, ...]  # Один элемент на каждый час работы
```

**Важно:** Количество элементов в `CLIENT_FLOW` должно совпадать с `WORK_END_HOUR - WORK_START_HOUR`

### Добавление/изменение кассиров:

```python
CASHIERS = [
    {'id': 0, 'simple_time': 120, 'auto_time': 75},  # секунд на клиента
    {'id': 1, 'simple_time': 140, 'auto_time': 70},
    # Добавьте своих кассиров здесь
]
```

### Изменение количества касс:

```python
NUM_SIMPLE_REGISTERS = 4   # Количество простых касс
NUM_AUTO_REGISTERS = 3     # Количество автоматических касс
```
